CTV Playground

Locking Transaction

OP_CTV (as described in BIP-119) locks a Bitcoin transaction to only a certainly enumerated set of outputs. Using the tools on this page, you can play with that on Signet or Regtest using Bitcoin Inquisition. Using these tools, you will create a CTV address and lock Bitcoin into, then create a spending transactions which will spend to only those outputs committed below.

First, describe the future outputs you wish your transaction to commit to. They should take the form:
address1:1.2btc
address2:4323sats
You can use any combination of addresses and amounts (in sats or BTC) that you like. After you lock the Bitcoin into these outputs, you will NOT be able to send your Bitcoin anywhere else.

Then, enter the Input Index of the CTV transaction. This is the vout of the CTV locking transaction. Usually it will be 0, but it will depend on which output your wallet software puts the CTV output.

Lastly, enter the network you wish send to. At this point, you should only use Signet or Regtest.
