CTV Playground

Locking Transaction

OP_CTV (as described in BIP-119) locks a Bitcoin UTXO to only be spent by a certain enumerated set of outputs. Using the tools on this page, you can play with that on Signet or Regtest using Bitcoin Inquisition. Using this tool, you will create a CTV address and lock Bitcoin into it, then you will create a spending transactions which will spend to only those committed outputs.

First, describe the future outputs you wish your transaction to commit to. They should take the form:
address1:1.2btc
address2:4323sats
You can use any combination of addresses and amounts (in sats or BTC) that you like. After you lock the Bitcoin into these outputs, you will NOT be able to send your Bitcoin anywhere else.

Then, enter the Input Index of the CTV transaction. This is the vout of the CTV locking transaction. Usually it will be 0, but it will depend on which output your wallet uses for the CTV output.

Lastly, enter the network you wish send to. Currently the CTV soft fork is only active on signet and regtest.
